File Metadata Microservice

View File Metadata

POST /api/fileanalyse

NOTE: This is a DEMO site. Files uploaded are not stored in any way, merely file metadata is processed and returned.

Example Usage:

To view file metadata:
  • Submit a file using the 'View File Metadata' form above.
  • Alternatively, submit a POST request to /api/fileanalyse with a body containing a multipart/form-data encoded field of 'upfile' containing the file to be analysed.

Example Output:

After uploading a file, file metadata will be returned in JSON format, with the following form:

{
  • "name":"logo4.png",
  • "type":"image/png",
  • "size":29193
}
